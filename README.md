# Read Me

Using PyTorch, Pandas, and NumPy I created a neural network model to predict All-NBA status. Using 20,000+ historical NBA player stats the model can accurately predict at ~98%.

Leveraged Flask and RESTful APIs to develop a full-stack website where users can input stats and receive real time feedback.

Used React and CSS to create the frontend.
